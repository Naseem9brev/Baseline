# Chrome Web Store assets

Everything needed to publish Baseline to the Chrome Web Store.

- **[SUBMIT.md](SUBMIT.md)** — step-by-step submission guide (the parts that need your
  Google account + the one-time $5 fee).
- **[LISTING.md](LISTING.md)** — listing copy (name, description), permission
  justifications, and the screenshot/asset checklist.
- **[PRIVACY.md](PRIVACY.md)** — the privacy policy text (also served at `/privacy` on the
  landing site).

Build the upload package from the repo root: `npm run zip` →
`.output/baseline-<version>-chrome.zip`.
